\begin{frame}
\frametitle{Short Summary}

Our objective : draw inferences about the data!
\newline

Steps:
\begin{enumerate}
\item select appropriate family of models,
\item estimate parameters of model,
\item check for goodness of fit to the data,
\item use fitted model to enhance understanding of data generating mechanism
\end{enumerate}
\pause

Examples of (4):
\begin{enumerate}
\item prediction/forecasting  
\item testing hypotheses (is trend parameter positive?)
\item simulate more data from fitted model
\end{enumerate}
